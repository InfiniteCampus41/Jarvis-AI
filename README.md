# Jarvis-ai-beta
beta version of jarvis ai run through a node.js server


# Setup

Must Have SOX And Node.js Installed

Step 1: Click On The Code Icon And Click Download Zip

Step 2: Extract The Zip File

Step 3: Open Folder In Terminal And Run 
npm install express

Step 4: Go To enter.pollinations.ai and sign up

Step 5: Scroll Down To The Api Keys Section And Create A New Api Key And Copy It

Step 6: Edit index.html and change 
Put your api key here
